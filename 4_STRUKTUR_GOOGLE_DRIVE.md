# ğŸ’¡ TIPS PENGGUNAAN GOOGLE DRIVE

## DO's âœ…

1. **Upload Segera:**
   - Upload file segera setelah selesai
   - Jangan tunggu sampai akhir

   - Backup file penting setiap hari
   - Simpan juga di local storage

3. **Naming Convention:**
   - Ikuti aturan penamaan
   - Jangan asal kasih nama

4. **Organize:**
   - Simpan file di folder yang benar
   - Jangan campur-campur

5. **Update Progress:**
   - Update Google Sheets setiap hari
   - Kasih catatan jika ada kendala

6. **Komunikasi:**
   - Kasih tau di grup kalau upload file baru
   - Tag orang yang perlu review

7. **Version Control:**
   - Jangan hapus draft lama
   - Pakai versioning (v1, v2, v3)

---

1. **Jangan Hapus File Orang Lain:**
   - Kecuali lo koordinator
   - Atau sudah koordinasi

2. **Jangan Upload File Besar Sembarangan:**
   - Compress dulu jika perlu
   - Atau pakai link external (WeTransfer, dll)

3. **Jangan Lupa Sync:**
   - Pastikan file ter-sync
   - Cek koneksi internet

4. **Jangan Asal Edit:**
   - Koordinasi dulu sebelum edit file orang lain
   - Pakai Track Changes di Word

5. **Jangan Ignore Notification:**
   - Cek notifikasi Google Drive
   - Respond jika ada yang mention lo
